## **Nhóm 1: Box Model & Kích thước**

| **Thuộc tính**                                       | **Đơn vị**              | **Công dụng**                         |
| ---------------------------------------------------- | ----------------------- | ------------------------------------- |
| `width`, `height`                                    | px, %, em, rem, vw, vh  | Đặt chiều rộng, chiều cao phần tử.    |
| `min-width`, `max-width`, `min-height`, `max-height` | px, %, em, rem, vw, vh  | Giới hạn kích thước tối thiểu/tối đa. |

| `border-radius`                                      | px, em, %, rem          | Bo tròn góc.                          |
| `box-sizing`                                         | content-box, border-box | Cách tính kích thước phần tử.         |

## **Nhóm 5: Flexbox**

| **Thuộc tính**    | **Đơn vị**                                                              | **Công dụng**                   |
| ----------------- | ----------------------------------------------------------------------- | ------------------------------- |
| `display: flex`   | —                                                                       | Bật Flexbox cho container.      |
| `flex-direction`  | row, row-reverse, column, column-reverse                                | Hướng trục chính.               |
| `flex-wrap`       | nowrap, wrap, wrap-reverse                                              | Cho phép xuống dòng.            |
| `justify-content` | flex-start, flex-end, center, space-between, space-around, space-evenly | Căn chỉnh theo trục chính.      |
| `align-items`     | flex-start, flex-end, center, stretch, baseline                         | Căn chỉnh theo trục phụ.        |
| `align-content`   | flex-start, flex-end, center, space-between, space-around, stretch      | Căn chỉnh nhiều dòng flex.      |
| `order`           | số (int)                                                                | Thay đổi thứ tự item.           |
| `flex-grow`       | số (0–n)                                                                | Khả năng phóng to item.         |
| `flex-shrink`     | số (0–n)                                                                | Khả năng co nhỏ item.           |
| `flex-basis`      | px, %, auto                                                             | Kích thước cơ bản của item.     |
| `flex`            | grow shrink basis                                                       | Viết tắt cho 3 thuộc tính trên. |

| `column-gap`            | px, em, rem, %              | Khoảng cách giữa cột.                       |

| `place-items`           | align justify               | Gộp 2 thuộc tính trên.                      |
| `justify-self`          | start, end, center, stretch | Căn chỉnh riêng từng item theo chiều ngang. |

## **Nhóm 7: Animation & Transition**

| **Thuộc tính**               | **Đơn vị**                                      | **Công dụng**                     |
| ---------------------------- | ----------------------------------------------- | --------------------------------- |
| `transition`                 | property duration timing-function delay         | Viết tắt cho hiệu ứng chuyển đổi. |
| `transition-property`        | all, tên thuộc tính                             | Thuộc tính sẽ chuyển đổi.         |
| `transition-duration`        | s, ms                                           | Thời gian chuyển đổi.             |
| `transition-timing-function` | ease, linear, ease-in, ease-out, cubic-bezier() | Đường cong tốc độ.                |
| `transition-delay`           | s, ms                                           | Thời gian chờ trước khi chạy.     |
| `@keyframes`                 | —                                               | Định nghĩa hoạt ảnh.              |
| `animation-name`             | tên keyframes                                   | Chỉ định hoạt ảnh.                |
| `animation-duration`         | s, ms                                           | Thời gian hoạt ảnh.               |
| `animation-iteration-count`  | số, infinite                                    | Số lần lặp lại.                   |
| `animation-direction`        | normal, reverse, alternate, alternate-reverse   | Hướng chạy hoạt ảnh.              |
| `animation-timing-function`  | ease, linear, cubic-bezier()                    | Kiểu tốc độ chuyển động.          |
| `animation-fill-mode`        | none, forwards, backwards, both                 | Trạng thái trước/sau khi chạy.    |
| `animation-play-state`       | running, paused                                 | Bật/tạm dừng hoạt ảnh.            |

## **Nhóm 8: Transform & Effects**

| `opacity`             | 0–1                                                                                                   | Độ trong suốt.                                    |
| `filter`              | blur(px), brightness(%), contrast(%), grayscale(%), hue-rotate(deg), invert(%), saturate(%), sepia(%) | Hiệu ứng ảnh.                                     |
| `backdrop-filter`     | blur(px), brightness(%), contrast(%), …                                                               | Hiệu ứng nền phía sau.                            |
| `mix-blend-mode`      | multiply, screen, overlay, darken, lighten, difference, exclusion…                                    | Trộn màu phần tử.                                 |
| `isolation`           | auto, isolate                                                                                         | Kiểm soát trộn màu.                               |
| `mask`                | giá trị mask                                                                                          | Che phủ phần tử.                                  |
| `mask-image`          | url(), gradient                                                                                       | Ảnh mask.                                         |
| `mask-repeat`         | repeat, no-repeat, space, round                                                                       | Cách lặp mask.                                    |
| `mask-size`           | auto, cover, contain, px, %                                                                           | Kích thước mask.                                  |
